# ARTICLE-HUB

A web-application for article management

## Local setup setup

### Required

- Java 17
- Maven 3.6+
- PostgreSQL 11+ or some alternative
- docker

### IDE:

- Eclipse 2021-12 or newer with Lombok installed, or
- Visual Studio Code
- or some alternative

### PROFILES:

- docker
- local
- default (add your own configuration)
